# Community Science Museum

A multi-page website for the Community Science Museum, designed to inform and engage visitors. The site covers general information, exhibitions, special events, ways to get involved, and how to plan a visit.

## Features

- Responsive layout with a hamburger menu for mobile devices
- Multiple pages: Home, Explore, Special Events, Visit, Get Involved, and Contact
- Shared header and footer across all pages
- Custom CSS per page with Google Fonts integration

## Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (e.g. VS Code) if you want to make changes

## Getting Started

### Viewing the site

Clone the repository and open `index.html` directly in your browser:

```bash
git clone https://github.com/johnruud3/Semesterproject.git
cd Semesterproject
```

Then open `index.html` in your browser. No build step or server is required.

## Technologies

- HTML5

- Google Fonts (Poppins, Roboto)
